# UI 設計書

## 1. 文書目的

本書は、許容領域シミュレータの Streamlit UI の画面構成、入出力項目、スライダー連動仕様、可視化仕様を定義する。

参照:
- requirements-spec.md セクション 11
- data-model-design.md
- algorithm-design.md

## 2. 実装形態

- フレームワーク: Streamlit
- 単一ファイル: `src/app.py`
- 言語: 併記（英語ラベル + 日本語説明）

## 3. 画面構成

Streamlit のサイドバーと本体を使い、4 セクションに分割する。

```
┌─────────────────────────────────────────────────────┐
│ サイドバー                     │ メインエリア          │
│                               │                      │
│ [Section 1] 条件設定           │ [Section 4] 結果表示  │
│  - マージン                    │  - Pass/Fail         │
│  - 不確かさ                    │  - 最悪点             │
│  - 安全係数                    │  - 許容量テーブル      │
│                               │  - 散布図             │
│ [Section 2] 評価点設定         │  - 寄与分離           │
│  - 最遠点                     │                      │
│  - 遠位点群                   │                      │
│                               │                      │
│ [Section 3] 6DoF シミュレーション│                     │
│  - 6 軸スライダー              │                      │
│  - 基準状態モード              │                      │
└─────────────────────────────────────────────────────┘
```

## 4. Section 1: Condition Settings / 条件設定

サイドバー上部に配置。

### 4.1 マージン入力

| 表示ラベル | 内部変数 | 入力方式 | 範囲 | デフォルト | 単位表示 |
|-----------|---------|---------|------|----------|---------|
| Vertical Margin / 垂直方向マージン | m_z | number_input | 0 — 50 | 10.0 | mm |
| Long Margin / 長軸方向マージン | m_y | number_input | 0 — 50 | 10.0 | mm |
| Lateral Margin / 側方マージン | m_x | number_input | 0 — 50 | 10.0 | mm |

### 4.2 不確かさ入力

expander で折りたたみ表示。3 軸 × 5 成分 = 15 入力。

表示形式:

```
▶ Uncertainty Settings / 不確かさ設定
  ┌─ Lateral (x) ─────────────────────┐
  │ U_identify:      [1.0] mm         │
  │ U_surrogate:     [2.0] mm         │
  │ U_registration:  [1.0] mm         │
  │ U_intrafraction: [1.0] mm         │
  │ U_model:         [0.5] mm         │
  │ Total:           2.69 mm (自動算出) │
  └────────────────────────────────────┘
  ┌─ Long (y) ────────────────────────┐
  │ （同様）                            │
  └────────────────────────────────────┘
  ┌─ Vertical (z) ────────────────────┐
  │ （同様）                            │
  └────────────────────────────────────┘
```

デフォルト値（仮値）:

| 成分 | x (mm) | y (mm) | z (mm) |
|------|--------|--------|--------|
| U_identify | 1.0 | 1.0 | 1.0 |
| U_surrogate | 2.0 | 2.0 | 2.0 |
| U_registration | 1.0 | 1.0 | 1.0 |
| U_intrafraction | 1.0 | 1.0 | 1.0 |
| U_model | 0.5 | 0.5 | 0.5 |

### 4.3 安全係数入力

| 表示ラベル | 入力方式 | 範囲 | デフォルト |
|-----------|---------|------|----------|
| Safety Factor z / 安全係数 | number_input | 0 — 5 | 2.0 |

## 5. Section 2: Evaluation Points / 評価点設定

サイドバー中部に配置。

### 5.1 最遠点入力

```
Farthest Point / 最遠点
  Name:  [farthest_1]
  x (Lateral):   [0.0] mm
  y (Long):      [100.0] mm
  z (Vertical):  [50.0] mm
  Distance from ISO: 111.8 mm (自動算出)
```

### 5.2 遠位点群入力

```
Additional Points / 追加評価点
  [+ Add Point / 点を追加]

  Point 2: distal_ant
    x: [80.0]  y: [0.0]  z: [60.0]
    Distance: 100.0 mm
    [🗑 Remove / 削除]

  Point 3: distal_lat
    x: [100.0]  y: [50.0]  z: [0.0]
    Distance: 111.8 mm
    [🗑 Remove / 削除]
```

- 動的に追加・削除可能
- session_state でリストを管理
- 各点の distance_from_iso を自動表示

### 5.3 将来拡張

CSV アップロードボタンの配置場所を確保しておく。v1 では disabled またはコメントアウト。

| 表示ラベル | 内部変数 | 範囲 | 刻み | デフォルト | 単位 |
|-----------|---------|------|------|----------|------|
| Vertical | vertical | -50 — +50 | 0.1 | 0.0 | mm |

### 6.2 単独許容量の基準状態モード

    Lateral:   [0.0] mm
    Rotation:  [0.0] °
    Pitch:     [0.0] °
    Roll:      [0.0] °

## 7. Section 4: Results / 結果表示

メインエリアに配置。スライダー変更時に全体が再計算・更新される。

### 7.1 全体判定ヘッダー

```
┌──────────────────────────────────────────────────┐
│  ✅ PASS  (or  ❌ FAIL)                           │
│                                                   │
│  Worst Point: farthest_1                          │
│  Constraint Axis: Lateral (x)                     │
│  Max Consumption: Q = 0.85                        │
│  Mag: 2.3 mm (参考 / reference)                   │
└──────────────────────────────────────────────────┘
```

- Pass: 緑背景
- Fail: 赤背景
- 評価点 0 個: 黄色背景 + 「評価点を設定してください / Please add evaluation points」

### 7.2 評価点別結果テーブル

| Point | d_iso (mm) | Δx | Δy | Δz | C_x | C_y | C_z | Q_x | Q_y | Q_z | Status |
|-------|-----------|----|----|----|----|----|----|-----|-----|-----|--------|
| farthest_1 | 111.8 | ... | ... | ... | ... | ... | ... | 0.85 | 0.72 | 0.63 | ✅ |
| distal_ant | 100.0 | ... | ... | ... | ... | ... | ... | 0.78 | 0.65 | 0.70 | ✅ |

- Fail の行は赤ハイライト
- 最悪点の行は太字
- 小数点 2 桁表示

### 7.3 許容量テーブル

2 つのテーブルを並列表示（st.columns で 2 列）。

#### 条件付き許容量

| Axis | Current | Min | Max | Rem(-) | Rem(+) | Limit Point | Limit Axis |
|------|---------|-----|-----|--------|--------|-------------|------------|
| Vertical | 0.0 mm | -8.2 | +8.2 | 8.2 | 8.2 | farthest_1 | z |
| Long | 0.0 mm | -7.5 | +7.5 | 7.5 | 7.5 | farthest_1 | y |
| ... | | | | | | | |

#### 単独許容量

同じ列構成。基準状態モードを括弧で表示: `(zero_based)` / `(current_based)` / `(custom)`

### 7.4 3D 参考指標

```
Reference Indicators / 参考指標 (3D)
  Effective Displacement (3D): 3.2 mm
  Max Margin Consumption: 0.85
  Mag (translation norm): 2.3 mm
```

### 7.5 散布図

- X 軸: distance_from_iso (mm)
- Y 軸: max(Q_i,x, Q_i,y, Q_i,z)
- 各点をドットでプロット
- Q = 1.0 に赤い水平線（マージン境界）
- Plotly または Streamlit のネイティブチャート

### 7.6 寄与分離バー

各評価点について、最悪軸の変位を並進寄与と回転寄与に分離して横棒グラフで表示。

```
farthest_1  [████████ trans ████ rot ██] 8.5 mm
distal_ant  [██████ trans ██████ rot ██] 7.8 mm
```

## 8. スライダー連動仕様

### 8.1 再計算トリガー

以下のいずれかが変更された場合、全計算を再実行する。

| 変更対象 | 再計算範囲 |
|---------|-----------|
| 6DoF スライダー | 全点判定 + 条件付き許容量 + 単独許容量（current_based のみ） |
| マージン | 全点判定 + 全許容量 |
| 不確かさ | 全点判定 + 全許容量 |
| 安全係数 | 全点判定 + 全許容量 |
| 評価点の追加/削除/変更 | 全点判定 + 全許容量 |
| 基準状態モード/値 | 単独許容量のみ |

### 8.2 更新される表示

スライダー変更時に以下がすべて更新される:
1. 全体判定ヘッダー（Pass/Fail, 最悪点, 制約軸）
2. 評価点別結果テーブル
3. 条件付き許容量テーブル
4. 単独許容量テーブル（current_based の場合）
5. 3D 参考指標
6. 散布図
7. 寄与分離バー

## 9. 表示書式

### 9.1 数値書式

| 項目 | 小数桁 | 単位 |
|------|--------|------|
| 座標・変位・マージン | 2 | mm |
| 不確かさ | 2 | mm |
| 許容量 | 2 | mm or ° |
| 消費率 Q | 3 | — |
| 距離 | 1 | mm |
| 角度 | 2 | ° |

### 9.2 色規約

| 状態 | 色 |
|------|-----|
| Pass | 緑 (#28a745) |
| Fail | 赤 (#dc3545) |
| 警告（評価点なし等） | 黄 (#ffc107) |
| exceeded | 赤背景 |
| Q > 0.8 | オレンジ（注意） |
| Q ≤ 0.8 | 緑 |
